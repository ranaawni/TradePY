#Point-Nine

#Features :
1: Trades page
2: Positions page

## Technologies

- Python 
- Flask
- React JS

## Start the app
- inside root of project we can run python :
1: source env/env/bin/activate
2: python app.py

- to run the frontend side :
 1: cd env/Web/point-nine
 2: npm install
 3: npm start
